# Platform for 5S processes

## Рабочая ссылка:

https://platform5s.vladislav-malchevskiy.ru/

## Инструкция по разворачиванию проекта:

### 1. Отклонируйте репозиторий:

```bash
git clone <URL вашего репозитория>
```

### 2. Установите зависимости:

```bash
npm install
```

### 3. Запустите проект:

```bash
npm run dev
```
